Problem

Designers, especially newbie design learners, don't have a channel to bond with relatively experienced designers.

This project is based on one of my previous research and design -- Design Mentorship App. Hence, this app has my user research and design as a basis.

Solution

An app that provides a friendly yet professional space for designers to connect and bond to ask for feedback on portfolio pieces or career advancement.

How It Works

1. Create profile and portfolio

swipe for designers that inspire you

2. Connect to other designers

to connect to your mentor

3. Send portfolio and chat

get professional feedback

Inspiration

To develop the app, I drew inspirations from other products. Behance provided inspirations for a clean layout for Profile and Search page. Dribbble provides me with inspirations to design displayed information on Profile page. Finally, UX.Coffee.Hours inspired me with its value proposition.

Development process

1. Sketches

2. Mid-fi wireframes

3. Flow charts

This medium-fidelity flow chart helps me build the logic of the app and help me grasp a high-level picture of how I can program the functions to support the features. Since I am my own designer and developer, this flow chart serves as a clarification tool for myself.


4. Data model

The data model is hosted on Google Firebase. Based on the flow charts, I was able to think through the structure of my backend data model for the app, which is crucial for me to think through functions I need to program to connect to Firebase.


5. Feature lists

To strategically build the application to deliver at least a Minimum Viable Product, I orchestrated each feature of the app based on the priority and closeness it is to the value proposition.
